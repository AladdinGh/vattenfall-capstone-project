# Milestone 1 — Create Your Own Repository

Now create your own GitHub repository in your own account.

Suggested naming pattern:
- `vattenfall-week9-capstone-yourname`

Your repository name should already communicate:
- theme
- week context
- ownership

## What to create now
- create the repository
- add a short initial `README.md`
- commit the first version

## What the first README should already say
At minimum:
- project theme
- business domains
- Day 1 focus
- week identity

## Why this milestone matters
A repository is not only storage.
It is the first communication layer of your engineering work.
