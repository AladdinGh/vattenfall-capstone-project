# Milestone 9 — Design the Governed Unity Catalog Structure

Your Day 1 design should already define the intended governed target.

Recommended example:
- catalog: `vattenfall_dev`
- schemas: `raw`, `refined`, `analytics`
- volumes: `landing`, `checkpoints`

## Why this matters
Day 1 is not only about GitHub.
It is also about the governed Databricks target the project will eventually use.

Document this in:
- your root README
- your docs
- your config
- your SQL setup file
