# Milestone 7 — Design the Python Package Layer

Inside `src/`, create:
- `utils/`
- `transforms/`
- `udfs/`
- `validation/`

Add `__init__.py` files.

## Why this matters
A senior-level project does not place all logic directly inside notebooks.

This layer prepares for:
- reusable helper functions
- modular transformations
- UDFs
- validation logic
- cleaner notebooks later in the week
