# Milestone 5 — Design the Sample Data Layout

Inside `sample_data/`, create domain folders:
- `market_prices/`
- `weather/`
- `grid_events/`
- `reference/`

## Why this matters
The capstone is multi-source.
Your raw file structure should reflect that immediately.

## Day 1 scale target
Prepare for approximately:
- 15 market price files
- 15 weather files
- 15 grid event files
- 5 reference files

You are building for scale, not for one toy file.
